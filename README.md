## Description

A starter project with NestJS + Nuxt.js

## Installation

```bash
$ npm install
```

## Running the app

```bash
# both Nest as Nuxt as one locally
$ npm run dev

# run only Nuxt
$ npm run dev:client

# run only Nest
$ npm run dev:server

# build (tested with aws elastic beanstalk)
$ npm run build
```
